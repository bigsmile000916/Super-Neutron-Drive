Neutron Beam Setup

Before you install Neutron Beam on your server, add a project and get your API Key. This is needed for your server installation.

Ubuntu/Debian

#Required Software
sudo -i
apt-get install build-essential python-dev libssl-dev libffi-dev
curl https://bootstrap.pypa.io/get-pip.py | python
pip install neutron-beam
exit

# Start Neutron Beam
nbeam start

# Enter your code directory
# Enter your username
# Enter your API Key

# Input your Server Address and Port to Neutron Drive
# If you have a firewall make sure the port is open to your browser's IP.

# Optional: Start Neutron Beam on boot
(crontab -l; echo @reboot `which nbeam` start) | uniq - | crontab -

Mac OS X

#Required Software
sudo -i
curl https://bootstrap.pypa.io/get-pip.py | python
exit

brew install openssl libffi pkg-config
export PKG_CONFIG_PATH=/usr/local/Cellar/libffi/3.0.13/lib/pkgconfig/
env ARCHFLAGS="-arch x86_64" LDFLAGS="-L/usr/local/opt/openssl/lib" CFLAGS="-I/usr/local/opt/openssl/include" pip install cryptography==0.6.1 --user
pip install neutron-beam --user
