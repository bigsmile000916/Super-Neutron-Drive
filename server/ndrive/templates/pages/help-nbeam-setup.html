Neutron Beam Setup

Ubuntu/Debian

#Required Software
sudo apt-get install python-pip build-essential python-dev
sudo pip install neutron-beam

# Start Neutron Beam
nbeam start

# Enter your code directory
# Enter your username

# Input your Server Address and Port to Neutron Drive
# If you have a firewall make sure the port is open to your browser's IP.

# Optional: Start Neutron Beam on boot
(crontab -l; echo @reboot `which nbeam` start) | uniq - | crontab -

Redhat/Cent OS/Other Redhat Based Distros

#Required Software
su
yum install python-setuptools gcc python-devel
curl https://bootstrap.pypa.io/get-pip.py | python
pip install neutron-beam
exit

# Start Neutron Beam
nbeam start

# Enter your code directory
# Enter your username

# Input your Server Address and Port to Neutron Drive
# If you have a firewall make sure the port is open to your browser's IP.

# Optional: Start Neutron Beam on boot
(crontab -l; echo @reboot `which nbeam` start) | uniq - | crontab -
